            - Defines some basic terms
                -elements - Building blocks of the html content
                Examples : title, head, body, heading, html

                -attributes - Give more information about element
                They are placed inside opening tags
                Examples:  href, id, class, style

                -tags - Denote elements 
                Example:  paragraph - <p></p>
                          container - <div></div>
